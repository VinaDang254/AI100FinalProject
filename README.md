AI Final Project – GenAI Debugging System
Overview
This project explores how AI systems behave when modified with intentional bugs. It uses GenAI as a reasoning partner to evaluate debugging reflections and improve problem‑solving skills.

Contents
- Google Sheet with 10 bug cases and reflections  
- PDF report describing learning outcomes  
- Python code for AI system experiments  

What I Learned
- How small changes in code or data can break AI systems  
- How GenAI can guide reasoning instead of giving direct answers  
- Importance of debugging, testing, and careful evaluation
